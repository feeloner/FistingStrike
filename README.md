## 🚀 Как запустить

### Вариант 1 — Один клик (рекомендуется)

1. Установи Java 21+ → [Скачать с adoptium.net](https://adoptium.net/)
2. Двойной клик по `FistingStrike.jar`Скачай последнюю версию → [последнюю версию JAR](https://github.com/feeloner/FistingStrike/releases/latest)
3. Двойной клик по `FistingStrike.jar`
4. Наслаждайся хуетой.

### Вариант 2 — Собрать из исходников (Установив предварительно Java 21+)

```bash
# Установи Java 21+

# Клонируй репозиторий
git clone https://github.com/feeloner/FistingStrike.git
cd FistingStrike

# Компилируй
javac -d bin src/game/*.java src/game/entities/*.java src/game/ui/*.java

# Собирай JAR
jar cfm FistingStrike.jar manifest.txt -C bin .

# Запускай
java -jar FistingStrike.jar
